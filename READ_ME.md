# MobileCLIP DL Project

## Setup Guide

### Prerequisites
- [Pixi](https://pixi.sh/) package manager
- Python 3.10

### Installation

1. **Enter the pixi environment:**
   ```powershell
   pixi shell
   ```

2. **Install mobileclip in editable mode:**
   ```powershell
   pixi run setup
   ```

### Usage

After setup, you can use mobileclip within the pixi environment:
```powershell
pixi shell
python your_script.py
```
